# 🎓 Spandan – AI-Powered Classroom Engagement Platform

Spandan is an AI-powered classroom engagement platform designed to make learning more interactive, accessible, and insightful. By combining speech recognition and Large Language Models (LLMs), Spandan transforms classroom discussions into meaningful learning experiences through real-time transcription, intelligent question generation, and participation tracking.

## 🚀 Features

* 🎙️ Real-time Speech-to-Text Transcription
* 🤖 AI-powered Question Generation
* 📄 Multi-format Input Support (Voice, Text, and PDF)
* 👨‍🏫 Teacher Dashboard
* 🎓 Student Interaction Portal
* 📊 Participation Tracking & Analytics
* 🔐 Secure Role-Based Access Control
* ⚡ Real-Time Classroom Engagement

## 📌 Problem Statement

Traditional classrooms often struggle with tracking participation, capturing discussions, and generating meaningful assessments from lecture content. Spandan addresses these challenges by leveraging AI to automatically transcribe classroom interactions and generate intelligent questions that improve engagement and learning outcomes.

## 💡 Solution

Spandan provides a unified platform where:

* Teachers can manage classrooms efficiently.
* Students can engage more actively.
* Classroom discussions are automatically transcribed.
* AI generates relevant questions from lectures.
* Participation insights are tracked in real time.

## 🏗️ Architecture
